Principais formatações:

Negrito/Destaque

Vou deixar o texto em negrito usando a tag B Negrito(Não semântica)

Vou deixar o texto em negrito usando a tag STRONG Negrito(semântica)

Itálico/Ênfase

Vou deixar o texto em Itálico usando a tag I Exemplo(Não semântica)

Vou deixar o texto com ênfase usando a tag EM Exemplo(semântica)

OBS:Gustavo recomenda que só sejam utilizados nos códigos HTML aas tags Semânticas, porque as não semânticas que tratam forma com o tempo se tornarão obsoletas porque que trata forma é o CSS3.😄
